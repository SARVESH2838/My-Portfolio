# Personal Portfolio Website

Responsive personal portfolio website with projects, certifications, resume-style sections, and interactive social navigation built using React and Tailwind CSS.

## 🚀 Features
- Responsive design for mobile, tablet, and desktop
- Interactive and modern user interface
- Project showcase section
- Certifications and achievements section
- Resume-style personal information layout
- Social media navigation (LinkedIn, GitHub, etc.)
- Smooth user experience with clean design

## 🛠️ Tech Stack
- React
- Tailwind CSS
- Vite
- JavaScript
- HTML5
- CSS3
